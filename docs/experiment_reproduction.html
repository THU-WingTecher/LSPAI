Experiment Reproduction

This page outlines the steps and resources for reproducing the LSPRAG experiments.

Back to Overview

Prerequisites

  • Install dependencies as described in README.
  • Prepare datasets for Java, Go, and Python projects.
  • Set up access to LLMs if running full pipeline; otherwise use provided logs.

Steps

  1. Run the LSP-backed retrieval to collect concise contexts.
  2. Generate unit tests and capture diagnostics for refinement.
  3. Measure line coverage and valid rate; export result CSVs.

Artifacts

  • Scripts in scripts/ and examples in experiments/.
  • Figures referenced in the overview page under docs/assets/.
