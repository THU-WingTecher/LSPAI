# Change Log

All notable changes to the "LSPRAG" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [Unreleased]

### Planned
- Additional language support
- Performance optimizations

## [0.2.0] - 2025-11-16

### Added
- AST structure support for enhanced code analysis
- Comprehensive logging system for debugging and tracking
- Configurable test file save path parameter
- Language Server testing command (`LSPRAG: Test Language Server`)
- Default test file save path now set to workspace root

## [0.0.2] - 2024

### Initial Release
- Basic unit test generation functionality
- Support for Python, Java, and Go languages
- Integration with Language Server Protocol (LSP)
- LLM provider support (DeepSeek, OpenAI, Ollama)
